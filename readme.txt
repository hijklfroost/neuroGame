En:
This project is non-commercial.
The idea of this project came to me when I was studying neural networks, and it consists in demonstrating the ability of a simple neural network to pass some kind of platformer game.
This project was supposed to be presented at one of the exhibitions of educational organizations in my city, but unfortunately, due to certain circumstances, this did not happen. But I decided to continue the project because of my own desire to complete it.
The intended end result of the project is that a person is given the opportunity to create a simple neural network, and then observe how it copes with the passage of a simple 2D platformer game.
I'm using Unity 2021 as my game engine.
On 03/29/2024: the project is not in any release version, I am experimenting with technologies to create.
